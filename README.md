# Hyper City

Hyper City is a hyper-local city services and business discovery platform that helps users find nearby service providers and local businesses using GPS-based location data. It also empowers neighborhood vendors to increase visibility, connect with customers, and grow through an easy-to-use digital storefront.

## Objective

Build a centralized digital platform to:

- Discover nearby services such as electricians, tutors, cafes, repair technicians, and more
- Use GPS location to show relevant local businesses and service providers
- Help local businesses increase visibility and connect with customers
- Provide a fast, friendly, and modern discovery experience for city residents

## Project Structure

- `hyper-city-backend/`: Backend API and server logic
- `hyper-city-frontend/`: Frontend application built with a modern web framework

## Tech Stack

- Backend: Node.js
- Frontend: Next.js

## Key Features

- GPS-based service discovery
- Local business listings and profiles
- Category-based search for services and shops
- Contact and booking options for nearby providers
- Responsive UI for mobile and desktop use

## Why Hyper City?

Hyper City is designed to bring local commerce online and make it easy for city residents to find trusted services near them. Whether someone needs a plumber, a tutor, or a neighborhood cafe, Hyper City simplifies the search and strengthens community businesses.

## How to Run

1. Open `hyper-city-backend` and install dependencies
2. Start the backend server
3. Open `hyper-city-frontend` and install dependencies
4. Start the frontend app

> Detailed commands depend on your preferred package manager and environment.
